PICT B.E. Courses

Course Structure

First Year Engineering (FE)

  • Semester 1

    Engineering Mathematics-I
    Focuses on calculus, differential equations, and complex numbers used in engineering.

    Basic Electrical Engineering
    Introduces DC/AC circuits, transformers, and basics of electrical machines.

    Engineering Physics
    Covers wave optics, quantum mechanics, and solid-state physics.

  • Semester 2

    Engineering Mathematics-II
    Includes Laplace transforms, vector calculus, and Fourier series.

    Programming in C
    Basic programming concepts like loops, functions, arrays, and structures using C.

    Engineering Mechanics
    Fundamentals of statics and dynamics with engineering applications.

Second Year Engineering (SE)

  • Semester 3

    Data Structures & Algorithms
    Teaches stacks, queues, trees, and sorting/searching algorithms.

    Computer Organization
    Explores CPU design, memory hierarchy, and instruction sets.

    Discrete Mathematics
    Foundations for computer science including logic, sets, and graphs.

  • Semester 4

    Operating Systems
    Concepts of processes, threads, memory management, and scheduling.

    Database Management Systems
    Relational databases, SQL, normalization, and transaction control.

    Theory of Computation
    Deals with automata theory, regular expressions, and Turing machines.

Third Year Engineering (TE)

  • Semester 5

    Computer Networks
    Study of OSI model, TCP/IP, routing, and network protocols.

    Web Technologies
    Frontend and backend web development using HTML, CSS, JS, PHP, etc.

    Software Engineering
    Principles of software development, life cycles, and quality assurance.

  • Semester 6

    Artificial Intelligence
    Search algorithms, knowledge representation, and basics of AI models.

    Cloud Computing
    Virtualization, cloud services (IaaS, PaaS), and deployment models.

    Cyber Security
    Basics of encryption, firewalls, and threat modeling.

Final Year Engineering (BE)

  • Semester 7

    Machine Learning
    Supervised, unsupervised learning, and model evaluation techniques.

    Big Data Analytics
    Analysis of large datasets using tools like Hadoop and Spark.

    Internet of Things
    Sensor networks, communication protocols, and IoT architecture.

  • Semester 8

    Project Work
    Capstone project applying core engineering concepts to solve a real problem.

    Electives
    Specialized topics like Blockchain, NLP, or Robotics depending on student interest.

    Industrial Training
    Internship in an industry setting to gain hands-on experience.
